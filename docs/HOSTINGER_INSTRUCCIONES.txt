GUÍA DE SUBIDA A HOSTINGER
==========================

Este paquete contiene tu sitio web listo para producción.

IMPORTANTE SOBRE LA IA:
-----------------------
El "Bot de IA" que creamos depende de `server.py` y `Ollama` (que corren en tu PC). 
En un plan de Hostinger "Web Hosting" o "Shared Hosting" (compartido), NO podrás ejecutar el bot de inteligencia artificial porque estos planes no permiten correr servidores Python ni modelos de IA pesados.

- Si subes estos archivos, la web funcionará perfecta (diseño, animaciones, subpáginas), pero el chat dará error de conexión.
- Para que el chat funcione en la web pública, necesitarías un VPS o conectar la web a una API externa (como OpenAI) en lugar de tu servidor local.

PASOS PARA SUBIR (Archivos Estáticos):
--------------------------------------
1. Entra a tu panel de Hostinger (hPanel).
2. Ve a la sección **Archivos** > **Administrador de Archivos**.
3. Entra en la carpeta **public_html** (Esa es la raíz de tu sitio).
4. Borra cualquier archivo "default.php" que haya ahí.
5. Sube el archivo `sitio_arcano.zip` que he creado.
6. Haz clic derecho sobre el zip subido y selecciona **Extract** (Extraer).
    - Asegúrate de extraerlo directamente en `public_html`, no en una subcarpeta.
7. Verifica que veas `index.html` suelto dentro de `public_html`.

¡Listo! Tu sitio estará online en tu dominio.
