LEER ANTES DE SUBIR
===================

El sitio web está listo para producción.

Archivos principales:
- index.html (Página principal en Español)
- index_en.html (Página principal en Inglés)
- styles.css (Estilos globales)
- script.js (Animaciones y lógica)
- services/ (Carpeta con las subpáginas de servicios)
- api/ (Backend PHP)
- admin/ (Panel de administración)
- database.sql (Archivo para crear la tabla en la base de datos)

Instrucciones de subida:
1. Sube TODO el contenido de la carpeta 'arcano-solutions' a la raíz de tu hosting public_html.
2. Crea una Base de Datos MySQL en tu panel de hosting (Hostinger/cPanel).
3. Importa el archivo 'database.sql' en tu base de datos (usando phpMyAdmin).
4. Edita el archivo 'api/config.php' con los datos reales de tu base de datos (host, nombre, usuario, contraseña).
5. (Opcional) Cambia la contraseña del admin en 'admin/dashboard.php'.

Notas:
- Se ha actualizado el copyright al año 2026.
- La funcionalidad de formulario REQUIERE que la base de datos esté configurada.
- Si pruebas localmente sin servidor PHP, el formulario mostrará una simulación (Demo).
